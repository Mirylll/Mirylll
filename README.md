## 🚀 Featured Projects

### 🏢 VNA-Project
> An internal admin & management system built during my internship

- Full RBAC (Role-Based Access Control) system with self-referencing Permission tree, Roles, and Titles
- JWT & OAuth2 authentication with OTP verification flows
- Enterprise, Industry, Province, and District management modules
- Soft-delete architecture with UUID primary keys
- Custom `PermissionsGuard` and decorators for fine-grained access control
- Polished UI with pagination, tree views, and permission-based UI hiding

**Tech Stack:** Next.js · NestJS · PostgreSQL
