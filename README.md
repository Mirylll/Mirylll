## Featured Projects

### VNA-Project
> Enterprise admin & management system — internship project

A multi-module internal platform built for managing enterprise hierarchies, user access, and organizational data. The system supports complex permission structures where roles are composed of fine-grained permissions organized in a self-referencing tree. Authentication is handled via JWT and OAuth2 with OTP-based verification for sensitive operations. The UI enforces access boundaries at the component level — users see only what their role allows, with no client-side hacks required.

Modules cover enterprise and subsidiary management, industry classification, and administrative geography down to district level. Data integrity is maintained through soft-delete patterns with UUID primary keys throughout.

**Tech Stack:** Next.js · NestJS · PostgreSQL

---

### Seminar Food Tour
> AI-powered food tour guide system — collaborative project with Bang

A multi-portal platform for managing and experiencing food tours with automated audio commentary. The system connects three types of users: admins who curate Points of Interest and generate AI-narrated audio guides, restaurant owners who manage their listings and menus, and tourists who explore the map and listen to multilingual commentary in real time.

Audio guides are generated using Google Gemini for translation across seven languages (Vietnamese, English, Chinese, Japanese, Korean, French, Spanish), then converted to speech and stored on Cloudinary. The user-facing app uses Leaflet for interactive map navigation with nearby restaurant discovery.

**Tech Stack:** React · TypeScript · Express · Prisma · PostgreSQL · Tailwind CSS · Framer Motion · Gemini API · Cloudinary · Leaflet

---

### Badminton Store
> Full-stack e-commerce platform for badminton equipment

A complete online store with a fully featured admin panel organized into functional sections: product catalog, sales and order management, inventory tracking, account management, and marketing tools. The frontend supports dark mode via React Context and is designed around a sidebar-first navigation pattern for efficient admin workflows. The backend is containerized with Docker for consistent local and production environments.

**Tech Stack:** React · TypeScript · Express · Prisma · PostgreSQL · Docker
